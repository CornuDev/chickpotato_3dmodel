# 3D 模型 - 像素雀薯

## 簡介
本倉庫包含使用 **MagicaVoxel** 製作的 3D 模型，引用 **Vtuber Akumu Ch. 阿庫姆** 的粉絲形象設計進行二創。
這些模型僅供相關社群成員自由使用。
